Model placeholders
==================

Download models manually and place them here:

- InsightFace "buffalo_l" (antelopev2) under `insightface/`
- Silent-Face anti-spoof `2.7_80x80_MiniFASNetV2.onnx` under `anti_spoof/`
- PaddleOCR English/Hebrew under `paddleocr/`

The Docker image mounts `./models` into `/models` for runtime use.
